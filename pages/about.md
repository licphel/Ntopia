# 关于本站

## Ntopia

Ntopia 是由 Licphel 创建的个人博客与知识社区，致力于构建自由、理性的技术交流空间。

## 技术架构

- 后端：Node.js + Express
- 数据库：SQLite (单一文件，轻量可靠)
- 模板：EJS (服务端渲染，复古风格)
- 部署：Docker + Nginx + certbot（一键部署脚本）

## 功能特性

- Markdown 编辑器 + 实时预览（分栏可拖拽）
- LaTeX 数学公式（KaTeX 渲染）
- 贴吧式嵌套评论系统
- 邮箱验证注册 / 改密 / 换绑 / 注销
- 等级经验系统（发文、评论、签到、被赞、被收藏）
- 文章许可协议（MIT / CC 系列 / GPL 等，每篇文章独立选择）
- 文章 Markdown 导出（含作者署名与原帖链接）
- 置顶文章
- 标签聚合页（点击标签或搜索框输入 #tag: 关键词）
- 私信通知
- 8 套主题换肤
- 权限分级管理（User / Mod / Admin / Super / Owner）
