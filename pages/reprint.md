## 转载本站内容

Ntopia 上的文章由作者自主选择许可协议。转载前请查看原文底部标注的许可类型，按对应协议要求操作。

## 各协议转载要求

| 协议 | 要求 |
|------|------|
| **All rights reserved** | 禁止转载，需联系作者获得书面授权 |
| **MIT** | 保留版权声明和许可声明即可自由使用 |
| **CC BY 4.0** | 署名原作者 + 标明原文链接 |
| **CC BY-SA 4.0** | 署名原作者 + 标明原文链接 + 相同方式共享 |
| **CC BY-NC-SA 4.0** | 署名原作者 + 标明原文链接 + 非商业使用 + 相同方式共享 |
| **CC0 1.0** | 无需署名，可自由使用 |
| **GPL 3.0** | 保留许可声明 + 开源传染 |

## 转载格式建议

转载时请在文首或文末附上：

> 本文转载自 [文章标题](原文链接)，作者 [作者名]，许可协议 [协议名称]。
